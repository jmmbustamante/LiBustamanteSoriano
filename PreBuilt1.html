Redragon K552

The Redragon K552 keyboard is a good low-budget starter keyboard that gets the job done. It has customizable RGB lighting which makes the keyboard look appealing and clear. Not only does it feel nice to use, but each key also clicks with a satisfying clack. It is also fairly responsive to each input with little to no issue, and has a fairly long lifespan. And at only Php 1600, the keyboard is very affordable given its high quality. Its only downsides include a lack of a number pad, however this may not be an issue for some. Overall, the Redragon k552 is a good, affordable, and reliable keyboard.
